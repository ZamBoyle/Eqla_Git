Créer un dépôt python pour le cours de Philip:
1. Allez dans le répertoire d'exercices python.
2. Faites git init
3. Faites git add .  (le point reprend tous les répertoires/fichiers et les sous-répertoires)
4. Faites git commit -m "Initial commit"
5. gh repo create ExercicesPython --public --confirm 
6. Faites git push -u origin master
